理想的居住环境灵感

Posted on 2020-03-01 | Post modified: 2020-03-01 | In 治愈系生活 | | Visitors:

很想记录下自己对理想居住环境的展望，就以细节体验的方式罗列自己的需求吧，后续也方便自己增加或者修改。：)

  • 非常喜欢北欧风格，比较干净，舒适：全屋以原木色和白色为主色调
  • 较多储物空间，有鞋柜的需求，有书柜需求，有通顶衣柜的需求
  • 希望坐在餐桌可以看到电视
  • 北卧室考虑榻榻米或者作为娱乐房或者空置
  • 喜欢在安静，遮光的环境下入睡，作息比较规律，卧室会放置投影仪
  • 朝九晚五上班族，晚上回家后会在客厅看电视，或读书听音乐
  • 房子自己住为主，父母也会来短时间居住，偶尔会请朋友来做客
  • 喜欢衣物都挂置起来，房间比较清爽的感觉
  • 有可能会有养狗需求，小型宠物犬
  • 如果厨房油烟问题可以解决的话 会倾向于半开放式厨房
  • 希望客厅有可以储物柜 放置数码产品
  • 希望有一个舒适的看书小角落
  • 希望有个书桌 放置电脑 数码周边
  • 会在客厅做运动，瑜伽或者椭圆机或者有氧操
  • 会考虑双开门冰箱，洗碗机，微波炉，烤箱
  • 对房子南北通透性的要求

Markdown

Posted on 2018-08-18 | Post modified: 2018-08-18 | In 技术分享 | | Visitors:

Markdown是什么？

它实际上是个非常简单、非常容易学习的语法。这个语法简单到每个人都可以在5分钟以内学会。应该是为数不多，你真的可以彻底学会的语言。
更重要的是，Markdown语法所有要素，是与写作的习惯一脉相承的，套用句俗语：仅为写作而生。

Markdown 的优点如下：

纯文本，所以兼容性极强，可以用所有文本编辑器打开。
让你专注于文字而不是排版。
格式转换方便，Markdown 的文本你可以轻松转换为 html、电子书等。
Markdown 的标记语法有极好的可读性。
